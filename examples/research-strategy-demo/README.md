# Research Strategy Demo

Run:

```bash
node dist/cli.js strategy opportunities --json
node dist/cli.js strategy rank --top 10 --json
node dist/cli.js strategy program --top 5 --json
node dist/cli.js strategy execute latest --max-cycles 3 --json
node dist/cli.js strategy reproduce-queue --json
node dist/cli.js strategy falsify-queue --json
```

The demo uses existing Sovryn corpus and memory artifacts. It does not require a
new GitHub repository.
